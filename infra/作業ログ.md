### 環境変数の準備（fish の例）

```fish
function load_vercel_tokens
    set -gx BW_SESSION (bw unlock --raw)
    set -gx TF_VAR_vercel_api_token (bw get password "Vercel API Token")
end
```

### 初回セットアップ

```bash
cd infra/terraform/vercel
load_vercel_tokens
terraform init
terraform plan
terraform apply
```

### 既存プロジェクトの import（必要な場合）
